\chapter{Keywords in Python}
\section{Python Keywords}

\textbf{Keywords} are reserved words in Python.
They have a special meaning and \textbf{cannot be used as variable names}.

Python keywords are \textbf{case-sensitive} and always written in \textbf{lowercase}.
